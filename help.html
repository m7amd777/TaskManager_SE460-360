How to Use the To-Do List App

Welcome to the To-Do List App! Here's a quick guide to help you get started:

  1. Adding Tasks: Use the input field at the top of the dashboard to add new tasks. Select a priority and click "Add Task".
  2. Editing Tasks: Click the edit icon next to a task to modify its details.
  3. Marking Tasks as Complete: Check the box next to a task to mark it as completed.
  4. Deleting Tasks: Click the delete icon to remove a task from your list.
  5. Filtering Tasks: Use the filter buttons to view tasks by priority or search for specific tasks using the search bar.
  6. Logout: Click the "Logout" button in the sidebar to securely log out of your account.

If you have any further questions, feel free to reach out to support!
